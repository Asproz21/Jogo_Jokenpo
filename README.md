Jogo Jokenpô Interativo

Bem-vindo ao Jogo Jokenpô Interativo, uma versão moderna, visual e divertida do clássico pedra, papel e tesoura! 🎮✨

👉 Acesse aqui: https://jogo-jokenpo-beta.vercel.app/

🚀 Sobre o Projeto

Este projeto transforma a lógica de um script em Python em uma interface web bonita, minimalista e interativa, trazendo uma experiência mais envolvente para o usuário.

A proposta é simples:

Jogar Jokenpô de forma divertida
🎯 Funcionalidades

✅ Interface moderna e minimalista
✅ Coleta do nome do jogador para personalização
✅ Sistema de escolha (Pedra, Papel e Tesoura)
✅ Resultado instantâneo da partida
✅ Experiência interativa e dinâmica

🧠 Regras do Jogo

O clássico que todo mundo conhece:

🪨 Pedra ganha da Tesoura
📄 Papel ganha da Pedra
✂️ Tesoura ganha do Papel
🛠️ Tecnologias Utilizadas
HTML
CSS
JavaScript
Deploy com Vercel
💡 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

Praticar transformação de lógica em interface visual

Criar uma aplicação simples, porém bem apresentada
